# Card Images

Place PNG or JPG images here for the card flip competition game.

Images should be:
- Square aspect ratio (recommended)
- PNG or JPG format
- Named descriptively (e.g., `star.png`, `robot.png`, `cat.jpg`)

The competition scene will randomly assign pairs of matching images to cards.

## Example Setup

For a 5x6 grid (30 cards = 15 pairs), you need at least 15 different images.

Sample images to add:
- Animals: cat.png, dog.png, fish.png, bird.png, etc.
- Objects: star.png, moon.png, heart.png, car.png, etc.
- Robots: robot1.png, robot2.png, robot3.png, etc.
