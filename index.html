What is the KRIA SOM Board?

The KRIA KV260 board that we are using for this bootcamp is an AI focused development kit for one of AMDs system on modules. A system on module (SOM) is a small integrated computing module that contains the essential components of a computer. This KRIA KV260 board was designed to accelerate AI vision applications. This makes it a very powerful and capable machine that can fit in the palm of your hand! These boards have a wide range of applications that include: robotics, industrial communication, security cameras and more. This make a perfect platform to get started working with AI.
